# Code of Conduct

We want `ossify` to be welcoming for maintainers, contributors, and curious first-time open source users.

## Expected behavior

- Be respectful and constructive.
- Give feedback with clarity and empathy.
- Assume good intent, then ask questions when something is unclear.

## Unacceptable behavior

- Harassment or discrimination
- Personal attacks
- Repeated bad-faith disruption
- Publishing private information without consent

## Reporting

Please contact the maintainers privately if you encounter harmful behavior.
